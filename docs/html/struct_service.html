Detailed Description

Представляет IT-услугу с ее свойствами.

Хранит название, ориентировочную стоимость, срок исполнения (в днях) и размер предоплаты для IT-услуги. Включает перегруженные операторы сравнения на основе стоимости, предоплаты и названия.
